# moviesImdb
Test for tap4

Before you start the app
------
* Create a class named **APIKEY.kt** 
* Copy and paste following code with your API KEY :
  ```kotlin
  class APIKEY{
    companion object {
        val TMDB_API_KEY = "YOUR API KEY";
    }
  }
  ```
 * These are required steps, app won't build without these.
